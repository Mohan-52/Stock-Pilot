================================================================================
                            API INTEGRATION
================================================================================

1. SEND OTP
   Endpoint: POST /api/auth/registration/otp/send?email=user@gmail.com
   Response: { "message": "OTP Successfully Sent" }

2. VERIFY OTP
   Endpoint: POST /api/auth/registration/otp/verify
   Request Body: { "email": "user@gmail.com", "otp": "148146" }
   Response: { "message": "Email Successfully Verified" }

3. REGISTER USER
   Endpoint: POST /api/auth/register
   Request Body: { "email": "user@gmail.com", "name": "John", "password": "..." }
